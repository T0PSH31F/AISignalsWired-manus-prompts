# Design Notes - AI Signals Wired

## Color Palette (Aura Theme Inspired)

Based on the Aura dark theme, we'll use a cyberpunk neon palette:

### Primary Colors
- **Background Dark**: `#15141b` (deep purple-black)
- **Background Secondary**: `#1e1d2b` (slightly lighter purple-black)
- **Neon Purple**: `#a277ff` (bright purple for accents)
- **Neon Cyan**: `#61ffca` (cyan for highlights)
- **Neon Pink**: `#ff6767` (pink/red for alerts)
- **Neon Green**: `#3de8b5` (green for success/wins)

### Semantic Colors
- **Text Primary**: `#edecee` (off-white)
- **Text Secondary**: `#6d6d6d` (muted gray)
- **Border**: `#2d2b38` (subtle border)
- **Success/Win**: `#3de8b5` (neon green)
- **Danger/Loss**: `#ff6767` (neon pink/red)
- **Warning**: `#ffca85` (warm orange)

## Design Style

**Neon Cyberpunk Theme:**
- Dark backgrounds with glowing neon accents
- Subtle fog/glow effects on hover
- Breathing animations on idle elements
- Sharp, modern typography
- High contrast for readability
- Glassmorphism effects for cards

## Typography
- **Headings**: Inter or Space Grotesk (bold, modern)
- **Body**: Inter (clean, readable)
- **Monospace**: JetBrains Mono (for numbers, prices)

## Key UI Elements
1. **Signal Cards**: Dark cards with neon borders, glow on hover
2. **Live Ticker**: Animated horizontal scroll with latest signals
3. **Performance Charts**: Neon line charts with glow effects
4. **CTA Buttons**: Glowing neon buttons with breathing animation
5. **Pricing Table**: Glassmorphism cards with tier highlights
